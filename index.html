IDeaScanner

IDeaScanner is both has a physical and application elements to it. We created a device that can scan RFID keys from IDs and sends it to our app. it also has a display that indicates that if you are logged or not.
The application is an attendance tracking tool that uses Depth First Search algorith to find and compare RFIDs to the one the device has scanned and return the following Name for the device to display. It also creates an excel file of which who has logged in using our device at that date.
